# Host 文件夹

这个文件夹包含了Host相关的脚本和配置文件，用于在宿主机上执行特定任务。

## 文件说明

- `requirements.txt` - Host环境所需的Python依赖包列表
- `job_mapping.txt` - 任务映射配置文件
- `restartflinkjob.py` - Flink任务重启脚本
- `monitor_flink.py` - Flink任务监控脚本
- `skills.py` - Host端技能定义文件
- `server.py` - Host服务端程序

## 功能

该文件夹中的脚本和程序负责在宿主机环境中执行特定任务，包括Flink任务的监控和重启、技能管理等。
